# Slime Rancher Planner REBORN
Simple web-app that helps you plan and organize your ranch in Slime Rancher game.

Forked by: Howie Stephens  
Based on [Slime Rancher Planner](https://github.com/baatochan/SlimeRancherPlanner) by: Bartosz Rodziewicz

## About the app
Displays a map of the Ranch with customization options for each farm plot, plus a "free-range" plot for each zone (designated with 'F'). Natural ponds (like at the docks can be set to a Pond type plot for Puddle slimes. 

Helps in planning out your ranch, reorganizing, or just sharing your layout.

Access the webapp here: [deathbypain.github.io/SlimeRancherPlanner-REBORN ![Slime Rancher Planner REBORN](https://img.shields.io/badge/website-live-brightgreen.svg)](https://deathbypain.github.io/SlimeRancherPlanner-REBORN)
